<h2>Open new image</h2>
  This button allows the user to choose and open a new image file. Any file format is acceptable by the system, regardless of its size and resolution.
  The chosen image is opened into the workspace region.
<br />
<h2>Open new file</h2>
  This button allows the user to choose and open a new file containing the features of at most 1 species to be classified.
  A window containing the contents of the file will be opened in the workspace region.
<br />
<h2>Run</h2>
  This triggers the classifiction and clustering of the input species. Once done, the results will be displayed in the workspace area. This includes the image, if present, features extracted, classification and clustering results.
<br />
<h2>Stop</h2>
  This stops the currently running classification and/or clustering operations.
<br />
<h2>Selection tools</h2>
  All tools necessary in choosing wich part of the image is to be classified/clustered is in this toolbar.
